# (1). 访问控制

# 1.public

公有的类成员可以在任何地方被访问
序列化后变成 属性名

# 2.protected

受保护的类成员可以被其本身以及子类和子类访问
序列号后变成 \x00*\x00属性名

# 3.private

私有的 1 类成员只能被其定义的所在类访问
序列化后变成 \x00类名\x00属性名
其中 \x00 表示空字符，但还是占用一个字符位置 (空格)
