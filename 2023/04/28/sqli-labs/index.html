# 怎么判断字符型或者数字型？

数字型:id=2 和 id=3-1 返回界面相同
原理：说明 2-1 被解析了，旁边没有引号
字符型:id=2 和 id=2a 页面一样就行
原理：MySQL 等号两边类型不一致会发生强制转换，字符串 -> 数字
