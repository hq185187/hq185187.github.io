# 一。什么是原型对象

我们知道在 JavaScript 中，一个已经存在构造器的对象中是不能添加新的属性的，但是可以通过 prototype 继承添加

# 为什么？

所有的 JavaScript 对象都会从一个 prototype (原型对象) 中继承属性和方法.JavaScript 对象有一个指向一个原型对象的链，当访问一个对象的属性时，不仅在该对象上搜寻，还会搜寻该对象的原型，以及该对象原型的原型，直到找到属性或者末尾.
