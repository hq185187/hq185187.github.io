# 简历

XML 被用来传输和存储数据，焦点是数据的内容，HTML 被用来显示数据，焦点是数据的外观
XML 标签没有被预定义，可以自定义标签
XML 不会做任何事情
XML 可以把数据从 HTML 中分离：当需要在 HTML 中显示动态数据时，将数据存储在独立的 XML，再通过几行 JavaScript 代码就可以读取一个外部 XML 文件，更新网页的数据内容

# 语法

  1. XML 标签对大小写敏感， <Message> 与 <message> 是不一样的
  2. XML 必须正确嵌套 (标签闭合顺序)
  3. XML 的属性值必须加引号
  4. 实体引用，"<" 放在 XML 元素中会报错，解析器把它当作新元素的开始，有 5 个预定义的实体引用
  1. XML 的注释和 HTML 相似
  1. XML 的空格会被保留，不会将多个空格合并为一个
  2. 外部 DTD 文件引用:
