# 原理

发生在应用程序解析 XML 输入时没有禁止外部实体的加载，导致可加载恶意外部文件，造成文件读取，命令执行，内网端口扫描，攻击内网网站，发起 dos 攻击等。xxe 漏洞触发的点往往在可以上传 xml 文件的位置，没有对上传的 xml 文件进行过滤，导致可上传恶意 xml 文件
