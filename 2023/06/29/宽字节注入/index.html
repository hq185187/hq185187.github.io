# 先介绍几个函数

  1. addslashes：在预定义字符（单引号，双引号，反斜杠，NULL）前加反斜杠
  2. mysql_real_escape_string：转义 SQL 语句中使用的字符串中的特殊字符 ( \x00,\n,\r,\,',",\x1a )

# 原理

窄字节：字符大小为一个一个字节
宽字节：字符大小为两个字节
所有英文默认占一个字节，汉字占两个字节
常见的宽字节编码：GB2312,GBK,GB18030,BIG5,Shift_JIS

当数据库用 GBK 编码而 PHP 用 UTF-8 编码，当防止 SQL 注入的上面的函数发挥作用，在单双引号前加了反斜杠 \ 。假如我们在单双引号前加一个字符，使其和斜杠组合成一个汉字，从而保留单双引号，使其发挥作用。
注意：
添加的字符 ASCII 要大于 128，两个字符才能组合成汉字，因为前一个 ascii 码要大于 128，才到汉字的范围 ，这一点需要注意，经典的有 % df

例子：Sqli-labs32 关
输入 1‘会被转义成 1\'
输入 1% df’%23 就可以成功回显
